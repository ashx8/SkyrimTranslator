# Skyrim Translator：天际翻译工具

本项目是基于 Godot 4.5 (C#) 开发的《上古卷轴5：天际》专用翻译工具（辐射4/新维加斯/星空有限支持，没经过太多测试），给懒人提供一些新的翻译方式？。

---

## 系统要求

- Godot 4.5 或更高版本
- .NET SDK 8.0 或更高版本
- Windows 10/11 操作系统

## 快速开始

### 构建与运行
1. 克隆本仓库
2. 使用 Godot 编辑器打开 `project.godot`
3. Godot 会自动生成 `.sln` 和 `.csproj` 文件
4. 点击编辑器中的运行按钮即可启动
5. 打包：点击 `项目` -> `导出` 选择对应平台

## 核心功能

### 分片词典系统
词典按首字母自动分片存储（a-z，#），大幅提升检索效率。支持用户词典与模组词典分离，自动覆盖词典中检索到的条目。

### 批量导入导出
可导出待翻译文本清单，交给网页AI批量翻译后导回。

导出方式有以下几种：
- 右键标签页可导出标签页内容
- 按住Ctrl多选标签页来导出
- 多选条目来导出

### 批量导入模组
可批量导入模组进行翻译，翻译后的模组会自动添加[已应用]的标记来识别模组是否已被翻译。

### 全局搜索功能
可搜索软件内所载入的所有模组条目（由B友nibiru2013提出）。
也可以搜索除排除中文外的其他内容（用于搜索未翻译内容）。

### 导入Strings文件作为词典
可导入游戏原Strings文件作为原文，第三方汉化如和光团队汉化后的Strings作为译文。

### 图书预览（由为主的祥和提出）
自动检测BOOK DESC字段显示图书预览按钮。

### 备份还原
翻译后会自动以日期格式备份原文件，并可在软件内手动选择需要还原的日期。

### 背景图设置（由为主的祥和提出）
没啥用的功能.....

### 关联NPC对话跟踪（由为主的祥和提出）
自动关联NPC对话条目，在翻译界面右侧显示当前NPC的所有对话，便于上下文统一。支持通过NPC名称快速跳转。

### 批量应用相似条目（由为主的祥和提出）
如good 1、good 2、good 3等，会弹出提示窗口进行批量应用（不更改后缀）。

### 自定义扫描字段
可以自定义需要扫描的字段以及不需要扫描的（防止不能翻译的被自己翻译了？）。
