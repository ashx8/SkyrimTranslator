# Skyrim Translator：天际翻译工具

本项目是基于 Godot 4.5 (C#) 开发的《上古卷轴5：天际》专用翻译工具，给懒人提供一些新的翻译方式？。

---

## 核心功能

### 分片词典系统
词典按首字母自动分片存储（a-z，#），大幅提升检索效率。支持用户词典与模组词典分离，自动覆盖词典中检索到的条目。

### 批量导入导出
可导出待翻译文本清单，交给网页AI批量翻译后导回。

### 图书预览（由为主的祥和提出）
自动检测BOOK DESC字段显示图书预览按钮。

### 关联NPC对话跟踪（由为主的祥和提出）
自动关联NPC对话条目，在翻译界面右侧显示当前NPC的所有对话，便于上下文统一。支持通过NPC名称快速跳转。

### 批量应用相似条目（由为主的祥和提出）
如good 1、good 2、good 3等，会弹出提示窗口进行批量应用（不更改后缀）。
